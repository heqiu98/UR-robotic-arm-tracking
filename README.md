# UR-robotic-arm-tracking
初步跟踪实验<br />
UR_robot_arm: UR5<br />
Camera: realsense D435i<br />
基于ROS控制机械臂平滑跟踪移动的Apriltag标志物,保持机械臂末端与tag平面平行,两者固连坐标系的z轴相对并在同一直线上.<br />
后续对控制算法进行修改，使得跟踪更快速、平滑<br />
程序在论文发出后上传.<br />
